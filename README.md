# docker-prestashop-phpmyadmin
Docker install : PrestaShop and PhpMyAdmin

Install :
- Open terminal
- ```docker-compose up```

URL PrestaShop :
- front [localhost:8080](localhost:8081)
- host : mariadb
- db : prestashop
- user : root
- password : mycustompassword

URL PhpMyAdmin :
- [localhost:8081](localhost:8081)
  - login : root
  - pwd : mycustompassword
